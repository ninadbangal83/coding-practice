# 🎨 HTML & CSS Learning Hub

Welcome to your HTML & CSS Learning Hub! This workspace is designed to take you from structural basics to high-end CSS layouts (Flexbox, Grid, and Animations).

---

## 📂 Folder Structure

* **`interview_cheat_sheet.html`** - ⚡ **THE ULTIMATE CHEAT SHEET**: High-velocity CSS recipes specifically tailored for 30-minute machine coding interviews!
* **`practice.html`** - Your active playground. Write your HTML markup here!
* **`practice.css`** - Your active styling playground. Write your custom CSS here (already linked in `practice.html`)!
* **`ex01_basics.html`** - Semantic HTML5 tags, headings, text formatting, buttons, and structural layouts.
* **`ex02_box_model.html`** - Understanding Margins, Paddings, Borders, Width/Height, and `box-sizing: border-box`.
* **`ex03_flexbox.html`** - Modern one-dimensional layouts (centering, distribution, rows vs. columns).
* **`ex04_grid.html`** - Two-dimensional layout grids (columns, auto-fitting, responsive cards).
* **`ex05_animations.html`** - Hover transitions, micro-interactions, and complex `@keyframes` animations.
* **`ex06_positioning.html`** - Relative, Absolute, Fixed, and Sticky coordinates.
* **`ex07_responsive.html`** - Designing for Mobile, Tablets, and Desktops with @media queries.
* **`ex08_forms.html`** - Styling modern form inputs, focus states, placeholders, and buttons.
