# Three.js Basic Learning Project

This project is located in the root of the `ninad` folder.

## 🛠️ Components
- `index.html`: Entry point.
- `style.css`: Modern styling.
- `main.js`: Three.js scene logic.

## 🚀 Key Learning
- Scene setup, Camera positioning, and Rendering.
- Lights and Materials.
- Animation loops and User input binding.
